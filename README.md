# comfyui-api

#### 介绍

对comfyui工具接口的二次封装，并且可视化控制comfyui工作流执行，目的是降低comfyui入手门槛和增加更多玩儿法

#### 软件架构

comfyui本地部署程序+api调用接口

ruoyi前后台管理框架（SpringBoot+Vue前后端分离的Java快速开发框架）


#### 开发计划

- [ ] 基于ruoyi框架增加一个简单的文生图页面
- [ ] comfyui工作流管理模块
- [ ] 成果图片管理模块+对接七牛云oss托管平台
